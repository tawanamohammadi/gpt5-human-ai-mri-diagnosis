
# Results Directory

This directory stores the output generated from the analysis scripts and notebooks in `/src` and `/notebooks`.

## Contents

This can include:
-   **CSV/JSON files**: Containing quantitative metrics (sensitivity, specificity, etc.).
-   **Tables**: Formatted tables for the manuscript.
-   **Plots and Visualizations**: Saved figures and plots that are not part of the final manuscript but were used during analysis.
-   **Model Outputs**: Raw text outputs from AI model interactions for logging and verification.

Files in this directory are considered artifacts and can be regenerated by running the code in the repository.

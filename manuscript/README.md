
# Manuscript Directory

This directory contains all files related to the research manuscript.

## Contents

-   **Preprint**: The version of the manuscript submitted to preprint servers like Zenodo or arXiv.
-   **Drafts**: Working drafts of the manuscript (e.g., `.tex`, `.docx` files).
-   **Figures**: High-resolution figures and charts used in the paper.
-   **Rebuttal_Letter.md**: The draft of the rebuttal letter addressing peer review comments.
